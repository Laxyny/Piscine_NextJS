# Frontend, Backend et Full-stack dans les frameworks JavaScript

## Objectif de l’exercice
Comprendre le rôle principal des frameworks JavaScript et savoir les classer selon qu’ils sont :
- Frontend
- Backend
- Full-stack

---

## Rappel (à lire avant l’exercice)

### Frontend
Framework utilisé pour créer l’interface utilisateur :
- HTML
- CSS
- Interactions dans le navigateur

---

### Backend
Framework utilisé pour gérer :
- La logique serveur
- Les API
- La sécurité
- L’accès aux données

---

### Full-stack
Framework capable de gérer :
- Le frontend
- Le backend  
dans un même projet.

---

## Énoncé de l’exercice

Vous disposez ci-dessous d’une liste de frameworks JavaScript.  
Pour chacun d’eux, indiquez s’il est principalement utilisé pour :
- le Frontend
- le Backend
- le Full-stack

**Consignes :**
- Mettez un **X** dans la ou les cases correspondantes
- Un framework peut avoir **une seule case cochée**
